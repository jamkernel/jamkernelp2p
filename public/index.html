🔐 Núcleo P2P · Funcionalidades (v1.0)

El núcleo jamkernelp2p es la base de todo el ecosistema. Está diseñado para ser ultraligero, portable y fácil de integrar en cualquier proyecto que requiera comunicación descentralizada.

🔐 Cifrado Militar

  • AES‑256‑GCM con autenticación
  • Derivación PBKDF2 (60.000 iteraciones)
  • Sal dinámica por sala
  • Identidad ECDSA P‑256 nativa

🛡️ Seguridad Integrada

  • Anti‑DoS por rate limiting (12 msg/seg)
  • Lista negra automática
  • Sanitización de entradas
  • Blacklist de peers maliciosos

🧠 Purga de Memoria

  • Eliminación forense de claves en RAM
  • Garbage Collection forzado
  • Nullificación de referencias

💾 Almacenamiento Local

  • IndexedDB con mutex antibloqueo
  • Backoff exponencial en fallos
  • Límite de cola (1000 items)
📖 Documentación completa del núcleo

🚀 Motor Híbrido · Node.js + Rust (v2.0 - En desarrollo)

El motor híbrido es la evolución natural del núcleo P2P. Añade una capa de alto rendimiento en Rust para tareas que requieren baja latencia, acceso a hardware y ejecución paralela, manteniendo la flexibilidad de Node.js para la orquestación.

🧠 Agentes IA Autónomos

  • Ejecuta agentes de IA directamente en el dispositivo
  • Toma decisiones en milisegundos sin depender de la nube
  • Ideal para asistentes personales, trading bots y monitoreo

⏰ Scheduler / Cron

  • Define tareas con expresiones cron
  • Ejecuta funciones en Rust de forma automatizada
  • Perfecto para mantenimiento, reportes y procesos batch

🔧 Tools Nativas

  • Registra funciones en Rust como herramientas
  • Llámalas desde Node.js o desde otros agentes
  • Extiende el sistema sin reiniciar

🧩 Bridge Node.js ↔ Rust

  • Comunicación bidireccional en tiempo real (WebSocket)
  • Compartir memoria y estado entre ambos motores
  • Lo mejor de ambos mundos: flexibilidad + rendimiento
